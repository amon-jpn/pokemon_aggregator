# Game Watch - ゲームグッズ最新情報 RSS

GAME Watchの「ゲームグッズ」カテゴリから、ポケモン以外の記事を自動抽出したRSSフィードです。

## 📡 RSSフィードURL

```
https://[あなたのユーザー名].github.io/[リポジトリ名]/gamewatch_goods.xml
```

## ✨ 特徴

- ✅ ゲームグッズカテゴリのみ抽出
- ✅ ポケモン関連記事を除外
- ✅ 1日4回自動更新（0:00, 6:00, 12:00, 18:00）

## 🔧 仕組み

1. GAME Watch公式RSSフィード（公開情報）を取得
2. ゲームグッズカテゴリでフィルタリング
3. ポケモン関連を除外
4. 新しいRSSとして公開

## ⚖️ 法的について

- 公式RSSフィード（公開情報）を利用
- タイトルとリンクのみ使用（本文のコピーなし）
- 1日4回のアクセス（サーバー負荷ほぼゼロ）
- 非商用・個人利用

## 📖 使い方

1. FeedlyなどのRSSリーダーを開く
2. 上記URLを登録
3. 完了！
